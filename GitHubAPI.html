Using Javascript

This project was also my introduction to Javasscript and Ajax. When one of the "commits" buttons is pressed, the url for that specific repo is sent back to a Javascript function. It is then sent to the controller via Ajax, and a call is made to GitHubs API. It sends back a list of all of the recent commits to the javascript function in JSON format. I built the HTML table inside of the javascript function, and then injected it onto the HTML page.

Figuring Out APIs

This project was my introduction to APIs. We were tasked with using GitHubs API to pull information about our personal Repos onto our own page. I pulled some basic info like my name, email and stuff like that. Then I pulled the list of repos that are public and contained on my profile. All of this info was populated in a ViewModel and sent back to the view. I used a foreach loop to print out the list of repos.
