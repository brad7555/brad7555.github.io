Writing My Own Database

For this project, I was tasked with creating a website for the coaches of a fake highschool track and field league. They wanted to be able to add their team, and individually add players, and be able to select a specific team to add them to. This is where I had to create my own database, and make sure that the player to team relationship was correct before I moved on. The relationship was, each athlete can have only 1 team, and each team can have many athletes.
